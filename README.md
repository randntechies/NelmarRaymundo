# Marmar Raymundo — Portfolio

A responsive, single-page portfolio built with Vue 3, Vite, and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Main edit points

- `src/App.vue` contains all portfolio copy, projects, navigation, and contact details.
- Update the `email` constant near the top of `src/App.vue`.
- Update the `projects` array to replace or extend project content.
- The design uses inline SVG and Tailwind utility classes only, so no external image or icon dependency is required.
