# Nelmar Raymundo — Static Portfolio

A dependency-free static portfolio using HTML, a precompiled Tailwind CSS file, and vanilla JavaScript.

## Deploy

Upload the full folder contents to any static host, including GitHub Pages, Netlify Drop, Cloudflare Pages, or Hostinger static hosting. The entry point is `index.html`.

## Structure

- `index.html` — all portfolio content and layout
- `assets/tailwind.css` — the precompiled Tailwind stylesheet
- `app.js` — mobile navigation, scroll-aware header, project modal, and footer year
- `assets/` — profile image and logo
- `favicon.ico` — tab icon

No Node.js installation, Vite project, package manager, Vue runtime, CDN request, or build command is required to run or deploy the site.
